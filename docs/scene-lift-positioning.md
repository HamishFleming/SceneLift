# SceneLift Positioning

SceneLift is the user-facing project name for this repository.

Recommended product line:

- `SceneLift`
- Subtitle: `AI background removal for creators and streamers`

Target audience:

- streamers who want a clean camera feed for OBS
- artists and creators who need easy cutouts for overlays, thumbnails, and edits
- non-technical users who want a simple live workflow instead of a model-focused tool

Brand guidance:

- keep the interface language plain and task-oriented
- avoid heavy ML jargon in user-facing copy where possible
- emphasize live webcam use, OBS compatibility, and transparent output

Compatibility notes:

- keep `bgremoval` as the primary CLI
- keep `bgremove` as the compatibility alias
- preserve existing Python API entry points unless a broader rename is explicitly requested
